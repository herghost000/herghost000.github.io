mysql 之 SET 类型

  • SET列最多可以有64个不同的成员
  • 一个表在其列ENUM和 SET列中可以有不超过255个独特的元素列表定义
  • 定义中的重复值将导致警告，或者如果启用严格SQL模式，则会发生错误。
  • SET列为NULL用IS NULL查询
  • SET列为空字符串用列名 = 0查询
  • 多选字符串数据类型，适合存储表单界面的“多选值”。
