外连接

  • 查询一个表有，另一个表没有的记录
  • 查询结果为主表中的所有记录
  • 从表中有匹配的则显示匹配的值
  • 从表中没有匹配的则显示null
  • 左外连接，left join 左边是主表
  • 右外连接，right join 右边是主表

左连接

aid title username
1 文章1 admin
2 文章2 admin
3 文章3 小明
4 文章4 NULL

右连接

aid title username
1 文章1 admin
2 文章2 admin
3 文章3 小明
NULL NULL Jack

交叉连接

  • 总行数为左表行数*右表行数
